Estudio de Caso

Sistema de Gestión de Créditos para Distribuidora

El Reto

Una distribuidora local gestionaba sus líneas de crédito de forma manual, lo que generaba errores, retrasos y frenaba su capacidad para crecer.

Mi Solución Creativa

Desarrollé un sistema web con Django en el backend y html, css y js en el frontend. Implementé un módulo de aprobación de créditos con reglas de negocio personalizadas y un dashboard analítico con Django para la generación de reportes.

El Impacto

Se logró una reducción del 80% en errores de captura y se aceleró el proceso de aprobación en un 50%, permitiendo a la empresa expandir su cartera de clientes.
